## 海螺影视主题

原创多功能MacCMS v10自适应主题

模板需要授权域名才能使用，并且确保你的环境可以安装SG11组件！！！一套主题授权一个顶级域名（含子域名），也就是授权baidu.com同时v.baidu.com也可以使用，同一个站点可绑定解析其他域名访问前台，无数量限制。
